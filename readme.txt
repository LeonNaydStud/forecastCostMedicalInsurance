Для запуска сервера необходимо:
1) Установить steamlit. Руководство по установке - https://docs.streamlit.io/library/get-started/installation#install-streamlit-on-windows
2) Во всех скриптах изменить пути ко всем файлам на сооветствующие вашим путям
3) Можете запустить сервер командой типа 'streamlit run server/main.py'
